# 整合清單 — 外部系統

## 內部整合（乙方自建）

| 系統 | 用途 | 介面方式 | 部署版本 |
|------|------|----------|----------|
| ChromaDB | 向量資料庫，儲存語料 embedding | Python client / REST | v1 ~ v3 |
| Ollama | 本機 LLM 推論（embedding + 生成） | HTTP REST API | v1 ~ v2b |
| FastAPI | 各 Agent 的 HTTP 服務層 | REST | v2a ~ v3 |

## 外部整合（第三方）

| 系統 | 用途 | 介面方式 | 部署版本 | 資料離境 |
|------|------|----------|----------|----------|
| Claude API（Anthropic） | 雲端 LLM 推論 | HTTPS REST | v3 | 是 ⚠️ |

## 注意事項

- Claude API 使用需甲方 IT 部門（王建國）另行審核資安政策
- Ollama 必須部署在甲方防火牆內，不得對外開放 11434 port
- ChromaDB REST API 僅限內部服務呼叫，不對外暴露
