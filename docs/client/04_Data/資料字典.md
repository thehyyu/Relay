# 資料字典

## 欄位定義

### Query（查詢紀錄）

| 欄位名稱 | 資料型態 | 說明 | 範例值 | 必填 |
|----------|----------|------|--------|------|
| query_id | string | 查詢唯一識別碼（UUID） | `q-20260423-001` | Y |
| query_text | string | 使用者輸入的自然語言問題 | `什麼是 CAP 定理？` | Y |
| timestamp | datetime | 查詢發出時間（UTC） | `2026-04-23T09:30:00Z` | Y |
| user_id | string | 查詢者識別（v3 才啟用） | `hubert` | N |
| response_id | string | 對應回答的 ID | `r-20260423-001` | Y |

### Document（語料庫文件）

| 欄位名稱 | 資料型態 | 說明 | 範例值 | 必填 |
|----------|----------|------|--------|------|
| doc_id | string | 文件唯一識別碼 | `doc-cap-theorem` | Y |
| title | string | 文件標題 | `CAP Theorem Overview` | Y |
| source_file | string | 來源檔案路徑 | `corpus/cap-theorem.md` | Y |
| content | string | 文件完整內文 | `（略）` | Y |
| chunk_index | int | 若文件被切片，此為段落序號 | `2` | Y |
| embedding | vector | 向量表示（維度依模型而定） | `[0.12, -0.34, ...]` | Y |

### SearchResult（搜尋結果）

| 欄位名稱 | 資料型態 | 說明 | 範例值 | 必填 |
|----------|----------|------|--------|------|
| result_id | string | 結果唯一識別碼 | `sr-20260423-001` | Y |
| query_id | string | 對應的查詢 ID | `q-20260423-001` | Y |
| doc_id | string | 命中的文件 ID | `doc-cap-theorem` | Y |
| similarity_score | float | 向量相似度（0~1，越高越相關） | `0.87` | Y |
| rank | int | 排名（1 = 最相關） | `1` | Y |

## 業務術語對照

| 業務術語（甲方用語） | 系統欄位 | 說明 |
|---------------------|----------|------|
| 知識庫文件 | Document | 匯入系統的語料，可為 Markdown、PDF 轉文字 |
| 搜尋命中段落 | SearchResult | Search Agent 回傳的相關文件段落 |
| 引用來源 | doc_id + source_file | 回答中顯示的來源文件資訊 |
| 查詢紀錄 | Query | Audit log 的基礎資料（Phase 1 可選） |
